Purpose Smallest routine possible to put variable width text at top left of screen.
Code 
	ld hl,$0000		;top left (0,0)
	ld (_penCol),hl		;load coordinates
	ld hl,display_text	;string to put up
	jp _vputs		;print string and return
Error While ld hl,$0000 may be the fastest way to zero hl, there's a smaller way that's five clock cycles slower. When speed's not an issue, try the following method to save a byte.
Fixed Code 
	or a			;clear carry (nc) for sbc
				; instruction
	sbc hl,hl		;hl=hl-hl (always
				; $0000 if nc)
	ld (_penCol),hl		;load coordinates
	ld hl,display_text	;string to put up
	jp _vputs		;print string and return
